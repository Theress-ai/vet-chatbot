# Veterinary Clinic Chatbot

## Overview
This project is an AI-powered veterinary chatbot that helps users ask questions and request appointments.

## Technologies Used
- FastAPI
- LangChain
- OpenAI API
- Vercel (deployment)

## Features
- Chat-based interaction
- AI-generated responses
- Simple web interface

## Deployment
1. Upload to GitHub
2. Import into Vercel
3. Add OPENAI_API_KEY in environment variables
4. Deploy
